# Módulo 3: Suas Primeiras Páginas Web

**Duração:** 12 horas
**Pré-requisitos:** Módulos 1 e 2 completos

## Objetivo do Módulo

Este módulo é a ponte entre programas Python de terminal e aplicações web. Os alunos aprendem como a web funciona e criam páginas bonitas e responsivas usando HTML e CSS.

## Estrutura das Lições

| Lição | Arquivo | Tópico | Duração |
|-------|---------|--------|---------|
| 3.1 | `1_como_web_funciona.html` | Cliente/servidor, HTTP, URLs | 1h |
| 3.2 | `2_html_basico.html` | Tags HTML, estrutura, semântica | 2h |
| 3.3 | `3_css_estilo.html` | Seletores, cores, flexbox | 3h |
| 3.4 | `4_responsivo.html` | Media queries, mobile-first | 2h |
| 3.5 | `5_formularios.html` | Inputs, validação HTML5 | 2h |
| 3.6 | `6_projeto_portfolio.html/.css` | Projeto integrador | 2h |

## Conceitos Aprendidos

### Lição 3.1 - Como a Web Funciona
- Modelo cliente-servidor
- O que são URLs e HTTP
- HTML, CSS e JavaScript
- Códigos de resposta (200, 404, 500)

### Lição 3.2 - HTML Básico
- Estrutura do documento HTML
- Tags semânticas (header, main, footer, nav)
- Títulos, parágrafos, listas
- Links, imagens, tabelas

### Lição 3.3 - CSS Estilo
- Seletores: tag, classe, ID
- Cores e tipografia
- Box model (margin, padding, border)
- Flexbox para layout
- Transições e hover

### Lição 3.4 - Design Responsivo
- Meta tag viewport
- Unidades relativas (%, rem, vw)
- Media queries
- Mobile-first design
- Grid responsivo

### Lição 3.5 - Formulários
- Tags form, input, label
- Tipos de input (text, email, password, etc.)
- Textarea e select
- Checkbox e radio
- Validação HTML5

## Entregável Final

**Portfólio Pessoal** com:
- Design moderno e profissional
- Seções: Início, Sobre, Habilidades, Projetos, Contato
- Formulário de contato funcional
- Totalmente responsivo (mobile, tablet, desktop)
- Pronto para publicar no GitHub Pages

## Para Visualizar

1. Abra qualquer arquivo `.html` diretamente no navegador
2. Ou use a extensão "Live Server" no VS Code

```bash
# Se tiver Python instalado, pode criar um servidor local:
cd modulo_03_paginas_web
python -m http.server 8000
# Acesse: http://localhost:8000
```

## Publicação no GitHub Pages

1. Crie um repositório no GitHub
2. Faça upload dos arquivos do portfólio
3. Vá em Settings > Pages
4. Selecione a branch "main" e salve
5. Seu site estará em: `https://seu-usuario.github.io/nome-repo`

## Dicas para Instrutores

- Mostre sempre o resultado visual antes do código
- Use as ferramentas de desenvolvedor do navegador (F12)
- Incentive experimentação - mudar cores, tamanhos, etc.
- O projeto final pode ser personalizado por cada aluno
