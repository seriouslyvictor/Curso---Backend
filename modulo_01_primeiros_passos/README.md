# Módulo 1: Primeiros Passos - Seu Primeiro Código

**Duração:** 12 horas
**Pré-requisitos:** Nenhum (iniciantes absolutos)

## Objetivo do Módulo

Levar estudantes que nunca programaram a construir um programa interativo completo. Cada lição produz algo que pode ser mostrado para amigos e família.

## Estrutura das Lições

| Lição | Arquivo | Tópico | Duração |
|-------|---------|--------|---------|
| 1.1 | `licao_1_1_primeiro_codigo.py` | Hello World, print() | 2h |
| 1.2 | `licao_1_2_variaveis.py` | Variáveis, tipos, f-strings | 2h |
| 1.3 | `licao_1_3_input_decisoes.py` | input(), if/elif/else | 2h |
| 1.4 | `licao_1_4_loops.py` | for, while, range() | 2h |
| 1.5 | `licao_1_5_listas.py` | Listas, append, remove, len | 2h |
| 1.6 | `licao_1_6_projeto_quiz.py` | Projeto integrador: Quiz | 2h |

## Como Usar

1. Execute cada arquivo Python na ordem (1.1, 1.2, etc.)
2. Leia os comentários explicativos no código
3. Complete os exercícios marcados com "EXERCÍCIO"
4. Tente os desafios extras se terminar cedo

## Para Executar

```bash
# Navegue até a pasta
cd modulo_01_primeiros_passos

# Execute uma lição
python licao_1_1_primeiro_codigo.py
```

## Conceitos Aprendidos

Ao final deste módulo, o aluno saberá:

- Usar `print()` para mostrar mensagens
- Criar e usar variáveis
- Usar f-strings para formatação
- Receber dados do usuário com `input()`
- Converter tipos com `int()` e `float()`
- Tomar decisões com `if/elif/else`
- Repetir ações com `for` e `while`
- Trabalhar com listas (adicionar, remover, percorrer)

## Entregável Final

**Quiz Interativo** com:
- Mínimo 5 perguntas
- Sistema de pontuação
- Mensagem de resultado final

## Dica para Instrutores

Comece cada aula mostrando o resultado final. Deixe os alunos verem o que vão construir antes de explicar o código.
